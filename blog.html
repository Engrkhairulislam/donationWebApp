    FAQ About Javascript DOM Element:

    What is the full form of DOM ?, and why we use DOM ?

    Full form of DOM is Document Object Model. We use DOM to dynamically access a HTML element of a HTML document/ page. in case of DOM system we can imagine whole HTML document is a Object. and this object has contains all of HTML element as a object property at different node.

    DOM is Dynamic for CSS id , class, if it's dynamic then how ?

    Yes DOM is completely dynamic for any type of operation by script handler programming language like as Javascript. to handle this type of dynamic operation DOM has many function and method. like cssClassList() can catch all of CSS class used in HTML document then you could done any CSS operation by add(), remove(), removeAttribute(), setAttribute() also many method function.

    can we could create dynamically HTML element by DOM ?

    DOM is completely dynamic model system. you can create, delete, hide, modify existing HTML element, by programming language. to do this type of operation DOM has many method, function. like document.create() method can crete any HTML element in HTML document. we can create any element in any depth of HTML document & after we can add proper position by append(), appendChild(), parentNode(), childNode() also many method function.

    how we can select HTML element by DOM ?

    we can select HTML element dynamically by DOM method like document.getElementById(), document.getElementsByClassName(), document.getElementsByName(), document.getElementsByTagName() after selecting any element we can do any type of operation by programming language.
